#D Shell
##About
The D Shell.  
This is the UNIX Shell Enviromnent, which is called "D Shell".  
It have Bash like syntax, but this is writteln in D language.  
  
  
##Installation
    % git clone https://github.com/alphaKAI/dsh
    % cd dsh
    % dmd main.d dshParser.d dshUtils.d
